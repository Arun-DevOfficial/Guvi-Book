"compilerOptions": {
"target": "ES6",
"module": "CommonJS",
"outDir": "./dist",
"rootDir": "./src",
"strict": true,
"esModuleInterop": true

  • module ‚Äì Specifies the module system (CommonJS for Node.js, ESNext for modern projects).
  • outDir ‚Äì Where compiled JavaScript files will be placed.
  • rootDir ‚Äì The directory where TypeScript source files (.ts) are stored.
  • strict ‚Äì Enables strict type checking (recommended for best practices).
  • esModuleInterop ‚Äì Allows compatibility with CommonJS and ES Modules.

üìå Step 1: Create a src Directory

5Ô∏è‚É£ Compiling TypeScript to JavaScript

To compile the TypeScript file, run:

This will generate a dist/index.js file that looks like this:

Now, run the JavaScript file using Node.js:
